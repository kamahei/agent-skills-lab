# Documentation

This directory contains human-facing operational documents for maintaining this repository.

## Included Documents

- `tool-mapping.md`
  - Maps each supported AI tool to its expected repository files and folders.
- `checklists.md`
  - Contains minimum review checklists for `AGENTS.md`, skills, agent files, and sample outputs.
- `env-example-policy.md`
  - Defines how `.env.example` files should be written and maintained in samples.

## Usage

- Read `tool-mapping.md` when choosing where a new file belongs.
- Read `checklists.md` before finalizing a new AGENTS, Skill, Agent, or sample.
- Read `env-example-policy.md` when a sample or template introduces environment variables.
