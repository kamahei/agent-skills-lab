# Gemini CLI Assets

This directory stores Gemini CLI-specific assets.

## Expected Contents

- `commands/`
  - Gemini CLI command definitions or reusable command assets.
- `skills/`
  - Gemini CLI skill files or skill wrappers.

## Repository-Level Instructions

- Use `GEMINI.md` at the repository root when you need persistent project instructions for Gemini CLI.
- Prefer shared source content in `.agents/` and only add Gemini-specific variants when necessary.
