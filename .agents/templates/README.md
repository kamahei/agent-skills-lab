# Templates

This directory contains reusable starter files for common deliverables in this repository.

## Included Templates

- `AGENTS.template.md`
  - General project-level `AGENTS.md` template.
- `CLAUDE.template.md`
  - Repository instruction template for Claude Code.
- `GEMINI.template.md`
  - Repository instruction template for Gemini CLI.
- `copilot-instructions.template.md`
  - Repository instruction template for GitHub Copilot.
- `copilot-agent.template.md`
  - GitHub Copilot agent template using `.agent.md` frontmatter.
- `copilot-skill.template.md`
  - GitHub Copilot skill template using `SKILL.md` frontmatter.
- `env.example.template`
  - Reusable environment file template for samples that need configuration placeholders.

## Usage Notes

- Copy the closest template and then edit the placeholders.
- Keep templates in English unless there is a strong reason not to.
- If a new pattern is reused more than once, promote it into a template here.
