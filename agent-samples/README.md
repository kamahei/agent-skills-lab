# Sample Outputs

This directory contains sample deliverables that can be copied into other repositories.

## Rules

- Create one folder per sample topic.
- Put the sample `AGENTS.md` at `agent-samples/<Topic>/AGENTS.md`.
- Keep sample files in English unless a request explicitly asks for another language.
- If a sample needs AI-specific variants, place them inside that sample's folder.

## Starter Template

- Use `agent-samples/_template/` as a starting point for new sample directories.
